    実行方法（実行ファイルの動かし方）

    実行するためのコマンドは以下の通りである。

    java version

    java -jar PrimeMinister.jar

    Python version

    make test

    Mac OS Xの場合には、アプリケーションPrimeMinister.appをダブルクリックするほうが簡便である。

    PrimeMinister.app

    また、openコマンドでPrimeMinister.appを開いても実行可能である。

    open PrimeMinister.app
